# rssspy

Firefox extension that checks whether the current tab exposes RSS/Atom feeds and lets you copy feed URLs.

## Load in Firefox

1. Open `about:debugging#/runtime/this-firefox`.
2. Click **Load Temporary Add-on**.
3. Select `/home/runner/work/rssspy/rssspy/manifest.json`.

Use the toolbar popup to scan the current page and copy a feed URL.
